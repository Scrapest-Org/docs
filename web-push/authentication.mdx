Most endpoints in the Web Push Service are protected and require authentication.

## API Key

To authenticate your requests, you must include the `x-api-key` header with a valid API key.

<ParamField header="x-api-key" type="string" required>
  Your unique API key generated via the `/auth` endpoint.
</ParamField>

## Generate API Key

To get started, you need to generate an API key. **Note:** There is a strictly enforced limit of **3 API keys** per entity.

```bash
curl -X POST https://webpush-production.up.railway.app/auth
```

### Response

```json
{
  "status": "success",
  "message": "API key generated successfully",
  "data": {
    "key": "7f8a9b..."
  }
}
```

### Example Request

<CodeGroup>

```bash curl
curl -X GET https://webpush-production.up.railway.app/health \
  -H "x-api-key: your-api-key"
```

```javascript fetch
fetch("https://webpush-production.up.railway.app/health", {
  method: "GET",
  headers: {
    "x-api-key": "your-api-key",
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

```python python
import requests

url = "https://webpush-production.up.railway.app/health"
headers = {
    "x-api-key": "your-api-key"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```go go
package main

import (
	"fmt"
	"io/ioutil"
	"net/http"
)

func main() {
	url := "https://webpush-production.up.railway.app/health"
	method := "GET"

	client := &http.Client{}
	req, err := http.NewRequest(method, url, nil)

	if err != nil {
		fmt.Println(err)
		return
	}
	req.Header.Add("x-api-key", "your-api-key")

	res, err := client.Do(req)
	if err != nil {
		fmt.Println(err)
		return
	}
	defer res.Body.Close()

	body, err := ioutil.ReadAll(res.Body)
	if err != nil {
		fmt.Println(err)
		return
	}
	fmt.Println(string(body))
}
```

```rust rust
use reqwest::header;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = reqwest::Client::new();

    let res = client
        .get("https://webpush-production.up.railway.app/health")
        .header("x-api-key", "your-api-key")
        .send()
        .await?
        .text()
        .await?;

    println!("{}", res);
    Ok(())
}
```

</CodeGroup>

### Error Responses

- **401 Unauthorized**: Missing or invalid `x-api-key`.
- **403 Forbidden**: API key limit reached (when generating keys).

```json 401 Unauthorized
{
  "status": "error",
  "error": "Missing or invalid x-api-key header"
}
```
