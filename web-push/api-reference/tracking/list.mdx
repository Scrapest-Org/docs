---
title: "List Tracked Users"
api: "GET https://webpush-production.up.railway.app/tracked-users"
---

Retrieve a list of all currently tracked users.

_Note: This endpoint returns the list of users tracked by your API key._

<RequestExample>
  ```bash curl
  curl -X GET https://webpush-production.up.railway.app/tracked-users \
  -H "x-api-key: YOUR_API_KEY"
  ```

```javascript fetch
fetch("https://webpush-production.up.railway.app/tracked-users", {
  method: "GET",
  headers: {
    "x-api-key": "YOUR_API_KEY",
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

</RequestExample>

<ResponseExample>
  ```json Response
  {
    "tracked_users": []
  }
  ```
</ResponseExample>
