---
title: "Untrack User"
api: "DELETE https://webpush-production.up.railway.app/track-user"
---

Stop tracking a user.

<Note>
  **Smart Untracking**: This endpoint implements smart untracking. The user will
  only be unfollowed on Twitter if no other API key is currently tracking them.
  Use with confidence.
</Note>

<ParamField body="username" type="string" required>
  The Twitter handle of the user to stop tracking.
</ParamField>

<RequestExample>
  ```bash curl
  curl -X DELETE https://webpush-production.up.railway.app/track-user \
    -H "Content-Type: application/json" \
    -H "x-api-key: YOUR_API_KEY" \
    -d '{ "username": "elonmusk" }'
  ```

```javascript fetch
fetch("https://webpush-production.up.railway.app/track-user", {
  method: "DELETE",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": "YOUR_API_KEY",
  },
  body: JSON.stringify({ username: "elonmusk" }),
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

</RequestExample>

<ResponseExample>
  ```json Response
  {
    "message": "Stopped tracking user elonmusk"
  }
  ```
</ResponseExample>
