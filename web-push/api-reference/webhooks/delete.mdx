---
title: "Delete Webhook"
api: "DELETE https://webpush-production.up.railway.app/webhook"
---

Remove the webhook registered to your API key.

<RequestExample>
  ```bash curl
  curl -X DELETE https://webpush-production.up.railway.app/webhook \
    -H "x-api-key: YOUR_API_KEY"
  ```

```javascript fetch
fetch("https://webpush-production.up.railway.app/webhook", {
  method: "DELETE",
  headers: {
    "x-api-key": "YOUR_API_KEY",
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

</RequestExample>

<ResponseExample>
  ```json Response
  {
    "message": "Webhook deleted successfully"
  }
  ```
</ResponseExample>
