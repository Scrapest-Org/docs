---
title: "Get Webhook"
api: "GET https://webpush-production.up.railway.app/webhook"
---

Retrieve the registered webhook for your API key.

<ResponseField name="data" type="string">
  The registered webhook URL, or null if none is registered.
</ResponseField>

<RequestExample>
  ```bash curl
  curl -X GET https://webpush-production.up.railway.app/webhook \
  -H "x-api-key: YOUR_API_KEY"
  ```

```javascript fetch
fetch("https://webpush-production.up.railway.app/webhook", {
  method: "GET",
  headers: {
    "x-api-key": "YOUR_API_KEY",
  },
})
  .then((response) => response.json())
  .then((data) => console.log(data));
```

</RequestExample>

<ResponseExample>
  ```json Response
  {
    "data": "https://your-webhook.com/endpoint"
  }
  ```
</ResponseExample>
